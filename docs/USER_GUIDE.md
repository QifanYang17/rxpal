# 此间有序 — 使用指南

> 您的贴心生活记录助手，帮您轻松管理健康、用药和家庭信息。

---

## 快速开始

### 第一次打开

1. 选择语言：**中文** 或 **English**
2. 输入您的**姓名**和**年龄**
3. 选一个喜欢的**头像**
4. 点击 **进入应用**

完成后会有一个简短的功能教程，带您认识五大功能。随时可以在「我的」页面重新查看教程。

---

## 五大功能

### 1. 首页 — 一眼看全天

打开应用就能看到：

- **今日服药进度** — 一目了然完成了多少
- **待办提醒** — 哪些药还没吃、什么时候该吃
- **最近就诊** — 上次看病的信息

点击提醒条目可以标记「已完成」或「跳过」。

---

### 2. 时间线 — 记录每一次就诊

**就诊记录：** 所有看病记录按时间排列，点击可以展开查看详情（医院、科室、医生、诊断等）。

**趋势分析：** 查看您的健康数据变化趋势。

- 点击顶部的指标按钮切换：**血压** / **血糖** / **心率** / **体重**
- 图表会自动画出变化趋势
- 点击 **+ 记录** 按钮可以手动录入当天的数据

#### 如何记录健康数据

1. 切换到「趋势分析」标签
2. 选择要记录的指标（如血压）
3. 点击右上角 **+ 记录**
4. 输入数值，点击 **保存**

---

### 3. 拍照识别 — 拍一拍就搞定

不用手动抄写病历！拍张照片，AI 帮您识别。

#### 识别病历/处方

1. 点击底部「识别」标签
2. 选择 **拍照识别** 或 **从相册选择**
3. 将病历或处方放在框内，确保文字清晰
4. 等待识别完成（会有暖心语录陪您等待）
5. 核对识别结果，如有错误可以直接修改
6. 点击 **确认并保存**

#### 识别药盒

1. 在「用药」页面点击 **扫描药盒**
2. 将药盒的用法用量面朝上拍照
3. 系统会自动识别药名、剂量、频率
4. 核对信息，点击 **确认并添加**

---

### 4. 用药管理 — 再也不怕忘吃药

#### 查看药物列表

- 每种药物显示：药名、剂量、频率、剩余天数
- 剩余不足 7 天会显示红色提醒

#### 查看时间表

切换到「时间表」标签，按时间段查看该吃什么药：

| 时段 | 时间 |
|------|------|
| 早晨 | 08:00 |
| 中午 | 12:00 |
| 傍晚 | 18:00 |
| 睡前 | 21:00 |

当前时段会高亮显示。

#### 添加新药物

**方式一：扫描药盒**（推荐）
1. 点击 **扫描药盒** 按钮
2. 拍照 → 自动识别 → 确认保存

**方式二：手动添加**
1. 点击 **手动添加** 按钮
2. 填写：药物名称、剂量、频率、服药时间、疗程、备注
3. 点击 **保存**

---

### 5. 我的 — 个性化设置

#### 修改头像

点击头像，从四个选项中选择。

#### 睡眠提醒

设置每晚提醒您休息的时间（默认 21:30）。

#### 字体大小

三档可选：**标准** / **大** / **特大**
适合不同视力需求。

#### 切换语言

在设置区域选择 **中文** 或 **English**，所有页面立即切换。

#### 家人绑定

1. 点击 **邀请家人**
2. 将邀请码发给家人
3. 家人输入邀请码即可绑定
4. 绑定后可以互相查看健康状况

## 常见问题

**Q: 退出应用后数据会丢失吗？**
不会。所有个人信息、药物列表、就诊记录都保存在手机本地，重新打开后自动恢复。

**Q: 拍照识别不准确怎么办？**
识别结果是可以修改的。识别完成后，直接点击需要修改的字段进行编辑，确认无误后再保存。

**Q: 可以同时管理多种药物吗？**
可以。通过「扫描药盒」或「手动添加」逐一添加，系统会自动按时间段整理成时间表。

**Q: 如何删除已添加的药物？**
目前可以在用药列表中管理药物状态。后续版本将增加删除功能。

**Q: 健康数据可以导出吗？**
导出功能正在开发中，后续版本支持。

**Q: 没有网络可以使用吗？**
大部分功能（查看药物、提醒、健康记录）离线可用。拍照识别功能需要网络连接。
